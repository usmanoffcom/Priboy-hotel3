# Статус проекта — сайт priboy1.ru

Репозиторий: [github.com/usmanoffcom/Priboy-hotel3](https://github.com/usmanoffcom/Priboy-hotel3). Продакшен: **https://priboy1.ru**.

## Выполнено (история)

1. **Структура**
   - Next.js App Router, TypeScript, Tailwind, shadcn/ui.
   - Отдельный репозиторий и деплой только для priboy1.ru.

2. **Метаданные**
   - `app/layout.tsx`: отель 3★, домен priboy1.ru, canonical / Open Graph по необходимости.
   - Бронирование: своя форма (не внешний виджет чужого домена).

3. **Навигация и блоки**
   - Шапка / подвал под бренд «Прибой», раздел развлечений вместо SPA-комплекса другого проекта.

4. **Данные**
   - `lib/rooms-data.ts` — номера и цены.
   - `lib/recreation-data.ts` — развлечения (бассейн, пляж и т.д.).

## Зоны внимания

- Страницы контента (о комплексе, контакты, цены, события, правила, реквизиты, политика) — периодически сверять с актуальной информацией отеля.
- Блог (`lib/blog-data.ts`) — уникальные тексты под 3★ и priboy1.ru.
- После правок формы бронирования проверять SMTP и Telegram на продакшене.
