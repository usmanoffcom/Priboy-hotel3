# Руководство по разработке — Прибой 3★

## Быстрый старт

**Требования:** Node.js 20+, pnpm 9+

```bash
git clone git@github.com:usmanoffcom/Priboy-hotel3.git
cd Priboy-hotel3
pnpm install
pnpm dev
```

В режиме разработки сайт: `http://localhost:3000`

**Сборка:**

```bash
pnpm build
pnpm start
```

## Переменные окружения

Скопируйте `.env.example` в `.env.local` и задайте значения. Пароли, токены бота и прочие секреты в репозиторий не коммитьте — на продакшене они задаются только в окружении сервера.

## Деплой

Скрипт `deploy.sh` в корне этого проекта. Нужны приватный SSH-ключ и хост — в **локальном** `.env.deploy` (см. `.env.deploy.example`) или в `.env.deploy` на уровень выше, если приложение лежит внутри папки вроде `Priboy3/Priboy-hotel3`. Секреты в git не коммитим.

```bash
# один раз, из каталога Priboy-hotel3:
cp .env.deploy.example .env.deploy
# при необходимости из обёртки: cp ../.env.deploy.example ../.env.deploy

./deploy.sh
```

Подробности: [`DEPLOY.md`](./DEPLOY.md) в этом же каталоге.

## Структура проекта (кратко)

- `app/` — маршруты Next.js (App Router), в т.ч. `api/booking`, `api/telegram`, страницы номеров, блога, бронирования
- `components/` — UI, форма бронирования, карточки номеров
- `lib/` — данные и утилиты (`rooms-data`, `blog-data`, `capacity` и др.)
- `public/` — статика

## Уведомления о бронировании

Заявки уходят на почту (SMTP) и в Telegram; имена переменных см. `.env.example`.

## Стили

Tailwind CSS, палитра: терракотовый / кремовый / белый. UI: [shadcn/ui](https://ui.shadcn.com/).
